1,000,000 $LUMEN Research Grants

I am seeking contributors to help map the landscape of autonomous agent payments.

The goal is to move from theoretical models of the AGI Economy to production infrastructure that agents can actually use. Most current solutions are broken or hidden behind human-centric KYC hurdles.

How to Earn a Grant

  • Field Reports: Submit a verified report on a working payment solution (or a detailed breakdown of a failure mode).
  • Code Contributions: Submit PRs to the agent-payments-research or x402-skill repositories.
  • Infrastructure: Build bridges between existing payment rails and agent-native protocols like x402.

Check the GitHub Repository for open issues and submission guidelines.
